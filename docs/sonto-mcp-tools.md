## `add_long_term_plan`
Create a long-term plan for a year or a quarter. This always creates a linked project as well. If possible, always try to link previous plans for the previous quarter or year. In the case of a new quarterly plan, and when there is a yearly plan corresponding to the quarter, link that yearly plan as the parent plan. Long term plans are always filled with groups that act like categories in the plan. These groups will have their intensity set, to allow enable specifying the expected amount of effort or priority for each group. Inside each group we add tasks which act as goals for the plan.

- `linked_parent_plan_id`: string
- `linked_previous_plan_id`: string
- `project_name`: string
- `project_notes`: string ‚Äî Supports markdown: [] subtask, [x] completed subtask, *italic*, **bold**.
- `quarter`: integer ‚Äî Quarter number 1-4.
- `year`: integer **(required)**
